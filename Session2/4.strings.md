# Data Type

- String

# concatination

- by new line

```
var x = "text 1"
    " text 2"
```

- by `+` with variables

```
var x = "text 1" + txt
```

- Interpolation

```
var x = "text 1 ${txt + ' ' + txt2}"
```

# scape

by backslash

```
var x = "text \"quote scape\""
```
